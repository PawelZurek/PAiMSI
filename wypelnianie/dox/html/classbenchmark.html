Detailed Description

Modeluje pojecie Benchmark.

Klasa sluzy do przeprowadzenia Benchmarku programu, tzn : -> wczytania 4 zestawu danych o ilosci elementow :
-> 10
-> 100
-> 1000
-> 10000
Na czterech klasach :
-> stos_lista
-> stos_tablica
-> kolejka_lista
-> kolejka_tablica
-> Na koniec zostana wyswietlone czasy kazdej akcji z osobna
