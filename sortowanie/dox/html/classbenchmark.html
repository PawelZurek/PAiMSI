Detailed Description

Modeluje pojecie Benchmark.

Klasa sluzy do przeprowadzenia Benchmarku programu, tzn : -> wczytania 6 zestawu danych o ilosci elementow :
-> 10
-> 100
-> 1000
-> 10000
-> 100000
-> 1000000
Trzema roznymi metodami :
-> Quick Sort
-> Merge Sort
-> Heap Sort
-> Na koniec zostana wyswietlone czasy kazdej akcji z osobna oraz czas calkowity
