Detailed Description

Modeluje pojecie graf. Klasa sluzy glownie do wykonania algorytmu a_star, czyli znalezienia najlepszej drogi miedzy dwoma punktami.

Constructor & Destructor Documentation

graf::graf ( int  rozmiar)

Konstruktor klasy graf.

Konstruktor jest parametryczny. Parametrem tym jest rozmiar, ktory dediniuje rozmiar siatki i tym razem ilosc wierzcholkow ( rozmiar^2 ). Nastepnie pozwala ustawic sciane w dowolnym miejscu oraz dowolnie wybrac punkty : poczatkowy i koncowy.
